# Datadog Monitoring — AWS EC2

## Overview
Real-time infrastructure monitoring 
of AWS EC2 using Datadog Agent.

## What is Monitored
- CPU Usage
- Memory Usage
- Disk Usage
- System Uptime
- Docker Containers
- Process Monitoring
- Log Management

## Setup Steps
1. Install Datadog Agent on EC2
2. Configure API key
3. Start agent service
4. View metrics on Datadog dashboard

## Tools Used
- Datadog Agent 7.x
- AWS EC2 (Ubuntu 26.04 LTS)
- Docker & Docker Compose
- Linux

## Metrics Captured
- CPU Usage: 0.35%
- Memory Usage: 84.31%
- System Uptime: 4.99 hrs
- CPU Idle: 99.48%
